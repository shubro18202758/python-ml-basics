## Key Topics Covered

### Statistics & Probability
- Probability distributions (Normal, Poisson, Exponential)
- Statistical hypothesis testing (Z-tests, Chi-square, Kolmogorov-Smirnov)
- Information theory (Entropy, KL-divergence)
- Bayesian inference and posterior updates
- Distribution fitting and anomaly detection

### Linear Algebra
- Distance metrics (Euclidean, Manhattan, Mahalanobis, Cosine)
- Matrix operations and transformations
- Eigendecomposition and SVD
- Covariance matrices and correlations
- Dimensionality reduction techniques

### Optimization
- Gradient descent and convergence
- Loss functions for classification
- Regularization techniques
- Cross-validation theory
